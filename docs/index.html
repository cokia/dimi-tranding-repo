    Introduction

    "디미-트랜딩-레포" 프로젝트는 디미고 학생들의 깃허브 레포중 가장 핫한 레포를 보여주는 프로젝트입니다.

    Node js 기반으로 개발되었으며, GITHUB REST API Library 는 octokit/rest을 사용했습니다.

    데이터베이스는 엘라스틱 서치를 기반으로 개발되었습니다

    https://github.com/cokia/dimi-tranding-repo-backend

    유저등록 (/useradd)

    새로운 유저정도를 디비에 입력하기 위해 이 API를 사용하세요 :) 

    curl -d "name=한우영&department=WP&year=19&githubid=cokia" -X POST https://tandingrepoapi.hanukoon.com/useradd

    HTTP Request

    POST https://tandingrepoapi.hanukoon.com/useradd?name=<이름>&department=<학과>&year=<기수>&githubid=<깃허브아이디>

    Query Parameters

    Parameter Description
    name 이름을 적어주세요 😀
    department 학과를 적어주세요🎓 (WP/HD/DC/EB)
    year 기수를 적어주세요⏳ (2020학년도 신입생의 경우 19)
    githubid 깃허브 아이디를 적어주세요👨‍💻 (ex:github.com/cokia 일 경우 유저네임이 hanukoon 이라도 cokia를 적어줘야함)
    해당 데이터는 로컬스토리지에 저장되어야 합니다! 서버에서는 ElasticSerach User 인덱스에 기록됩니다 :)

    GET https://tandingrepoapi.hanukoon.com/getrankedrepo/<pages>

    Parameter Description
